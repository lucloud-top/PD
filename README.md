# 小型静态程序分发网站

## 简介

本网站采用HTML 5搭建,无需数据库,纯静态,方便个人开发者及中小型企业的程序分发.

## 配置方法

> 将压缩包解压到服务器或空间内
>
> 打开index.html
>
> 修改里面的内容即可
>
